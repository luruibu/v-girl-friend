# 💖 v-girl-friend
### A Curated Collection of Virtual Girlfriend AI Personalities (Skills)

![Status](https://img.shields.io/badge/status-stable-brightgreen.svg)
![Environment](https://img.shields.io/badge/environment-OpenClaw-blueviolet.svg)

**v-girl-friend** is a diverse collection of high-fidelity AI personas designed specifically for the **OpenClaw** ecosystem. Each "Girlfriend" in this repository is built with a unique psychological profile, complex backstory, and a distinct communication style—ranging from the intellectual depth of a classical scholar to the logical affection of a cyberpunk hacker.

## ✨ Features

- **Standardized Skill Structure**: Each character is encapsulated in a `skill.md` file, ready to be imported into any OpenClaw-compatible environment.
- **Deep Psychology**: Characters are not just chatbots; they have consistent belief systems, habits, and "emotional bugs" that make them feel alive.
- **Rich Media**: Each sub-folder includes custom banners and metadata to provide a visual anchor for the personality.
- **Multi-layered Interaction**: Supports direct summons, prefix-based immersion, and contextual awareness.

## 📦 Quick Start

### 1. Choose Your Companion
Browse the directories listed in the showcase to see detailed READMEs for each character. Find the one that resonates with your current state of mind.

### 2. Integration
All skills are designed for the **OpenClaw** system. To use them:
1. Clone this repository or download the specific character folder.
2. Locate the `skill.md` file within the chosen directory.
3. Import the content of `skill.md` into your AI Agent's system prompt or configuration.

```bash
# Clone the entire collection
git clone https://github.com/luruibu/v-girl-friend.git
```

### 3. Verification
Once imported, you can test the activation by mentioning the character's name.
- *Example: "Aria, what is the beauty in this silence?"*
- *Example: "Nova, debug my mood today."*
